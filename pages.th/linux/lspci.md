# lspci

> แสดงอุปกรณ์ PCI ทั้งหมด
> ข้อมูลเพิ่มเติม: <https://manned.org/lspci>.

- แสดงรายการอุปกรณ์แบบย่อ:

`lspci`

- แสดงข้อมูลเพิ่มเติม:

`lspci -v`

- แสดง driver และ module ที่เกี่ยวข้องกับอุปกรณ์:

`lspci -k`

- แสดงอุปกรณ์ที่กำหนด:

`lspci -s {{00:18.3}}`

- แสดงข้อมูลในรูปแบบที่อ่านง่าย:

`lspci -vm`
